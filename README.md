# Маркетинговый дашборд с AI-аналитиком

Маркетинговый дашборд с AI-аналитиком. Данные из Excel, визуализация в браузере, AI отвечает на вопросы по данным.

## Структура проекта

```
├── data/           # Данные (Excel файлы, CSV, JSON)
├── components/     # Компоненты графиков и визуализаций
├── assets/         # Брендинг (логотипы, иконки, шрифты)
├── .agent/
│   └── skills/     # AI-навыки и промпты для аналитика
└── src/            # Исходный код приложения
```

## Возможности

- Загрузка данных из Excel/CSV файлов
- Интерактивные графики и диаграммы в браузере
- AI-аналитик отвечает на вопросы по загруженным данным
- Экспорт отчётов и визуализаций

## Запуск

```bash
npm install
npm run dev
```
